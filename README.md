# My Python Learning Projects

A collection of Python projects built while learning programming.

## Projects
- **Number Guessing Game** - A game with difficulty levels
- **Shopping List** - A list manager with file storage
- **Contact Manager** - OOP-based contact book with file storage
- **Exam Tracker** - Track school exams with SQLite database
- **Calculator** - A simple calculator for basic math operations
- **Weather App** - Get weather information using an API
- **Bank Account** - OOP-based bank account simulation
- **Currency Converter** - Convert currencies using exchange rates
- **Finance Manager** - Manage income and expenses with SQLite database
- **Flight Tracker** - Track flight information using an API
- **IP Geolocation** - Get location information from an IP address
- **NASA Photo** - Get and save NASA photo of the day
- **News Aggregator** - Get latest news from an API
- **OCR** - Extract text from an image
- **Password Manager** - Save and manage passwords
- **Telegram Bot** - A simple Telegram bot project
- **Translator** - Translate text using Python
- **Wikipedia API** - Search information from Wikipedia
## Skills learned
- Python basics, Variables, conditions, loops and functions, Lists and dictionaries, OOP, File I/O, SQLite, Working with APIs, Working with images and OCR, Git
